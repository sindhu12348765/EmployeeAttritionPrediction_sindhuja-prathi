Employee Attrition Prediction Using Gradient Boosting Classifier
Overview

This project predicts employee attrition using a Gradient Boosting Classifier. The model analyzes employee-related factors such as age, job satisfaction, monthly income, overtime, work-life balance, and years at the company to determine whether an employee is likely to leave the organization.

The Gradient Boosting Classifier effectively predicts employee attrition and identifies key factors influencing employee turnover. The model can assist HR departments in making data-driven decisions to improve employee retention.
